Software solutions and electronic devices for communication and exchange of information between contracting authorities and economic operators.<br/><br/>Additional Information:<br/>These tools can be used for, at least, these five situations:<br/><br/>1. The URL of the system from where to access the procurement documents;<br/>2. The URL of the system for the submission of tender documents;<br/>3. The URL of the system from where to download a tool to communicate with the Buyer;<br/>4. The URL of the system used by a Technique to allow Economic Operators to exchange information with the Buyer (e.g. eAuction and DPS Systems)<br/>5. Thee URL of the system used to exchange information between Buyer and EO for questions and clarifications;<br/><br/>Initially approved in 2019-02-05<br/>Reviewed in 2019-07-11 and now pending of approval by the WG<br/><br/><br/>
