A condition that needs to be answered for evaluation purposes.<br/><br/>Additional Information: <br/>1) A criterion without properties has an informative value, e.g. it describes what it is expected form a responder.<br/>2) A criterion can be refined through the specification of sequences of properties of type "information requirement" or "constraints".<br/>3) Properties can be grouped in sub-criteria.<br/><br/><br/><br/><br/>
